\# Webpage RAG Chatbot (Level-4)



A Retrieval-Augmented Generation (RAG) chatbot that trains on complete websites

using internal link crawling and answers questions using OpenAI.



\## Features

\- Full website crawling (internal pages)

\- TF-IDF + FAISS vector search

\- OpenAI-based answer generation

\- Streamlit Cloud deployment



\## Run Locally

```bash

pip install -r requirements.txt

streamlit run app.py
